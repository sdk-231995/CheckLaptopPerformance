Second hand laptop condition check



1. battery check Use Windows Battery Report (CMD)==> powercfg /batteryreport

2. dead pixel(Screen) test = https://lcdtech.info/en/tests/dead.pixel.htm

3. keypad test=> https://en.key-test.ru/

4. all port should be check

5. hard disk health => CrystalDiskInfo9_6_3

6. BIOS check (  window+r type msinfo32)  or cmd systeminfo (to check laptop age)

7. performance check by running application

8. Device manager =>  to check hardware component

9. go to TASK MANAGER => then to performance tab => see RAM,CPU others performance

10. slmgr.vbs/ dlv => window license

11. check ram/memory by running windows memory dianostic

12. HWMonitor to check laptop over heat
